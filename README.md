# Clinic Medical System - Backend

Sistema de gerenciamento de clínica médica com Laravel e Docker.

## Requisitos

- Docker
- Docker Compose

## Como executar

1. Clone o repositório
2. Execute: `docker-compose up -d`
3. Acesse: http://localhost:8000

## Serviços

- Laravel API: http://localhost:8000
- MySQL: localhost:3306
- ML API: http://localhost:8001

## Variáveis de Ambiente

As configurações do banco estão definidas no docker-compose.yml
